DOM :- Collection of ReadyMade Code.

DOM :- Its based on node tree which is created for HTML document.

everything written in HTML gets convereted into node.
All nodes -share parent , child and their sibling.
Root Node :- HTML
Any node can have n no of children node.
Any node can have only one parent node.
Few nodes can leaf node(leaf not) which have no child.
Nodes having common parent called sibling.


DOM tree refers to the HTML page where all the nodes are objects . There can be 3 main types of nodes in the DOM tree.
1. text nodes
2. element nodes
3. Comment nodes

Descendent nodes :- All nested elements , children , their children and so on.
